gfx++

Open source. Lightweight, portable and fast frontend application framework made with C++ and SDL2. 2D GUI graphics library. Experimental 3D library. Powerful and easy to use.

UI Components

button, label, linear layout, text field, dropdown, listbox, image, checkbox, message box, progress bar, scrollbar, tabs, table, and a global color palette). Backgrounds, borders and padding. Positioning with Cartesian coordinates (x/y) and scaling with width and height. Padding and layouts make it simple to prototype application interfaces. Perfect boilerplate code for a new desktop application.

Linear Graphing

Static & Realtime

GUI Event Handling

(list all mini comps) KeyDown, KeyUp, LoseFocus, MouseDown, MouseUp, MouseMotion, MouseWheel. Manage these events with ease using GItem and GuiCommanders.

3D (Experimental)

Render colored 3D objects (untextured). Right click to rotate camera and scroll to zoom.
